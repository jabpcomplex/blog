Primeros pasos para cambiarte a Linux

Si estas harto de qué se trabe o ralentice tu computadora con windows y no te alcanza para comprar una imac o simplemente quieres dejar atrás windows. Este post tiene como objetivo ayudarte a cambiar a linux. Linux es un sistema operativo de código abierto, aunque nadie lo utiliza como tal, sino que recurren a las distribuciones (distros) ya que, con ellas, se vuelve mucho sencillo de utilizar y cada distro (ubuntu, kubuntu, debian, azure, etc) tiene su toque personal.
